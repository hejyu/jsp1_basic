POST 방식 요청

POST 방식 요청은 form을 이용해서 동작합니다. Get은 파라미터를 url과 함께 전달하지만 POST 는 HTTP 프로토콜 메시지 body에 저장ㅇ하여 요청을 보냅니다. POST 는 서버 데이터의 추가/ 수정/ 삭제 와 같은 변경 작업에 주로 사용됩니다. POST 는 파라미터 값 인코딩 해주지 않습니다.
